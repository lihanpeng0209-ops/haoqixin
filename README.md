# 外国政要参访苏州 · 1971—2026

> 从「逛园林」到「直奔工厂」——56 年苏州外交足迹交互地图

🔗 **[在线访问](https://lihanpeng0209-ops.github.io/haoqixin/)**

---

## 项目简介

收录 1971 年至 2026 年间 66 个国家和地区的外国政要 79 次参访苏州记录，以交互地图、时间轴、筛选面板和数据图表，直观呈现苏州从"园林外交客厅"到"全球产业现场"的转型历程。

---

## 功能特性

- **交互地图**：Leaflet 深色地图，颜色编码三类来访者（元首·王室·部长），点击弹出详情及新闻链接
- **热力图**：按参访密度生成动态热力分布
- **年代筛选**：1970s—2020s 六档年代切换，图表面板联动
- **参访目的趋势**：堆积柱状图，展示园林文化 / 混合参访 / 产业经贸的占比演变
- **参访地点变迁**：堆叠柱状图，追踪园林古迹→产业园区→工厂企业的重心转移
- **自动叙事**：四段自动播放模式，讲述从西哈努克游园到 APEC 金鸡湖的时代故事
- **移动端优先**：触控友好，微信内可直接打开，一套代码自适应手机与桌面
- **加载优化**：OSM 瓦片 + CSS 深色滤镜（无白块），脚本延迟加载，BootCDN 国内加速
